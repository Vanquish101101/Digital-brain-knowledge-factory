# Knowledge Factory — каркас (LIGHT-профиль)

Инфраструктурный каркас технического бэкенда: векторный поиск (Qdrant), метаданные
(Postgres/pgvector), сырые файлы (MinIO), кэш/память (Redis). Граф (Kuzu) — встроенная
библиотека, поднимается позже вместе с `kf.py`, отдельного контейнера не требует.

Полная архитектура и профили — `../Файлы настройки проекта/Архитектура проекта. Настройка и Правила/core.md`,
`rules.md`, `ПЕРЕД-СТАРТОМ.md`. Текущий статус решений — `../Файлы настройки проекта/РЕШЕНИЯ-И-СТАТУС.md`.

## Статус: стек запущен, все 4 контейнера healthy (2026-07-11)

- Qdrant: http://localhost:6333/dashboard
- MinIO: http://localhost:9001
- Postgres: localhost:5432
- Redis: localhost**:6380** (не 6379 — занят чужим контейнером другого проекта)

Postgres хранится на именованном томе Docker (не на `E:`) — Windows/NTFS bind-mount не может
дать контейнеру строгие Unix-права, которые требует Postgres. Подробности и как при желании
перенести на `E:` — в `../Файлы настройки проекта/РЕШЕНИЯ-И-СТАТУС.md`.

Осталось перед первым запуском:

2. `docker compose up -d` и дождаться healthy у всех 4 контейнеров (`docker compose ps`).
3. Проверка глазами: Qdrant dashboard — http://localhost:6333/dashboard, MinIO — http://localhost:9001.

## kf.py — готов и работает (2026-07-11)

CLI на Python (uv, venv в `.venv/`), 45 тестов (TDD, RED-GREEN на каждый модуль),
часть — против живого стека (Postgres/Qdrant/MinIO), не моков.

Команды:
- `uv run python kf.py ingest [--source PATH]` — индексация (по умолчанию —
  `../Структура хранилища`). Идемпотентно (sha256, resume).
- `uv run python kf.py search "запрос"` — семантический поиск фрагментов.
- `uv run python kf.py ask "вопрос"` — поиск + связный ответ через OpenRouter, со ссылками.
- `uv run python kf.py stats` — сколько документов/чанков в базе.

Текущее состояние базы: 23 документа, 26 чанков (только `Структура хранилища`;
`Файлы настройки проекта` и `CLAUDE.md` пока не индексируются — можно добавить `--source`
отдельным запуском, если понадобится искать и по ним).

Исключено из индексации: `Закладки браузера — структура.md` (низкая ценность для поиска,
несоразмерный объём) — см. `kf/scope.py` (`EXCLUDED_FILENAMES`).
