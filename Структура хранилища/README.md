# Цифровой мозг

Личная среда для сбора идей, задач, знаний, целей, проектов и материалов, которые помогают думать, создавать и двигаться дальше.

## Быстрый вход

- [Навигация](Навигация.md) - главная карта пространства.
- [001 Входящие (Сырые данные)](001%20Входящие%20(Сырые%20данные)/README.md) - быстрый сбор мыслей, ссылок, задач и сырого материала.
- [002 Мышление (Карта личности)](002%20Мышление%20(Карта%20личности)/README.md) - интересы, хобби, ценности, сильные стороны, стиль мышления.
- [003 Знания](003%20Знания/README.md) - заметки, конспекты, модели, наблюдения.
- [004 Инструменты](004%20Инструменты/README.md) - софт, сервисы, шаблоны и рабочие связки, которыми пользуюсь.
- [005 Ресурсы](005%20Ресурсы/README.md) - книги, курсы, ссылки, инструменты.
- [006 Задачи (Цели)](006%20Задачи%20(Цели)/README_Задачи.md) - действия, списки, недельные фокусы; там же [цели](006%20Задачи%20(Цели)/README_Цели.md).
- [007 Проекты](007%20Проекты/README.md) - активные и будущие проекты.
- [008 Интересы (Увлечения)](008%20Интересы%20(Увлечения)/README.md) - хобби и увлечения вне рабочих направлений.
- [010 Архив (Неиспользуемое)](010%20Архив%20(Неиспользуемое)/README.md) - завершенное, замороженное и больше не актуальное.
- [Шаблоны](010%20Архив%20(Неиспользуемое)/Шаблоны/README.md) - формы для идей, задач, проектов и заметок.

## Основной принцип

Сначала фиксировать, потом связывать, потом превращать в действия.

1. Все новое попадает во входящие.
2. Раз в несколько дней входящие разбираются по папкам.
3. Знания связываются с целями и проектами.
4. Проекты превращаются в конкретные задачи.
5. Завершенное уходит в архив, но не исчезает.
